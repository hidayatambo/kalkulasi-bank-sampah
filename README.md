

### Instalation and Runing

- composer update
- npm install & npm run dev
- copy .env.example
- php artisan optimize
- php artisan migrate
- php artisan serve
- php artisan db:seed --class=UserSeeder
- php artisan db:seed --class=JenisSampahrSeeder
